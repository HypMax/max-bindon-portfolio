### Video Games Sales Analysis

- **Code:** [crypto-whale-alert-tracker](Whale_Alert_Tracker.py)
- **Description:** A Python script that captures, cleans, and categorizes tweets from the "whale_alert" Twitter account, which reports transaction behavior of major players (whales), creating a well-structured dataset. It lays the groundwork for uncovering trends, patterns, and correlations which can then be leveraged to explore connections between transaction behavior of major players (whales) and the price movements of a specific cryptocurrency coin. 
- **Skills:** Python programming, API integration, Data cleaning, Cloud storage management.
- **Technology:** Python, Pandas, Numpy, Tweepy (Twitter API), Google Cloud Storage.
- **Results:** A [relational database of transactions](whale_alert_tweets (SAMPLE OUTPUT).csv) updating every 24 hours and a separate [junk database](junk_whale_alert_tweets (SAMPLE OUTPUT).csv) of tweets which don't meet transaction criteria. This database is now ready to be used for more in-depth analysis uncovering trends, patterns, and correlations which can then be leveraged to explore connections between transaction behavior of major players (whales) and the price movements of a specific cryptocurrency coin.
